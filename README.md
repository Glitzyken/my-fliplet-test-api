## 💻 Running Locally

- Download the codebase from Github
- Run `npm install`
- Get `.env` variables [👉 here](https://drive.google.com/file/d/18yozDnUXOhnFG1fZ3ug-ioKSFMmcjOjZ/view?usp=sharing)
- Run `npm run dev` to start the local server
